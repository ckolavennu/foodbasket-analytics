# Foodbasket Analytics

Foodbasket Analytics is a data analytics project focused on analysing the cost of essential food items and understanding food affordability pressure through basket-level pricing, category contribution, and dashboard-based insights.

## Project Status

In Progress

## Problem Statement

Food prices are often discussed in broad terms, but people usually experience the impact through a full basket of essential items. This project aims to make that impact clearer by analysing how different food categories contribute to the total basket cost.

## Objective

To analyse the cost of a basic food basket, identify the food categories contributing most to basket cost, and present insights through an interactive dashboard.

## Planned Methodology

1. Collect or structure food price data for selected essential basket items.
2. Clean and prepare the dataset for analysis.
3. Calculate basket cost and category contribution metrics.
4. Build visualisations and dashboard views to communicate findings.
5. Summarise insights, limitations, and practical recommendations.

## Planned Tools

## Planned Outputs

- Cleaned food price dataset
- Exploratory data analysis
- Basket-level affordability metrics
- Interactive dashboard
- Final insight summary

## Project Scope

This project analyses the cost of a basic student food basket in Malaysia using item-level grocery price data. It focuses on total basket cost, category-level contribution, and basic affordability insights.

## Target User

Students and young adults in Malaysia who want to understand monthly grocery cost and food affordability pressure.
